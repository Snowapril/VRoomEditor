static RTMesh RTG.RTMesh.Create ( Mesh  unityMesh)
static

Factory function for creating an RTMesh instance from the specified Unity mesh. The function will return null if the RTMesh can not be created. Such a scenario occurs when the Unity mesh is not readable.

The client code is responsible for calling 'BuildTree' for the returned mesh.
