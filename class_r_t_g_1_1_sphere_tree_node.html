◆ DebugDraw()

void RTG.SphereTreeNode< T >.DebugDraw ( )

Allows the node to render itself for debugging purposes. The client code is responsible for setting up the rendering material.

This method is recursive and will draw the node's children also. Thus, it is enough to call this method for the root of a sphere tree in order to draw the entire tree.
